# Sources - Credential Access (Pack 01)

## Sumber 01
Judul: MITRE ATT&CK - Credential Access (T1xxx)
Tipe sumber: standard/best practice
Relevansi untuk Pack 01:
- Pemetaan teknik credential access untuk baseline deteksi dan hunting.
- Menetapkan taksonomi untuk spraying, brute force, dan stuffing.
- Referensi penamaan teknik dan dokumentasi playbook.
Catatan keandalan: primer, teknis, umum (non-vendor), stabil.

## Sumber 02
Judul: NIST SP 800-63B Digital Identity Guidelines
Tipe sumber: guidance/standard
Relevansi untuk Pack 01:
- Praktik autentikasi, rate limiting, dan lockout untuk mitigasi.
- Parameter kebijakan password dan kontrol MFA.
- Rujukan kontrol defensif di playbook.
Catatan keandalan: primer, teknis, formal.
